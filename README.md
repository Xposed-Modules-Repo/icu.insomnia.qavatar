## 功能：

- 从指定文件夹下自动上传QQ头像

  `/storage/emulated/0/Android/data/com.tencent.mobileqq/files/qavatar/`

- 支持从指定URL下载图片作为头像



## 初始化：

- 见QQ设置首页
- 或见QQ账号管理设置页
-
